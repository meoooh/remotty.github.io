  • 새로운 글을 작성하기 위해서는 아래와 같이 명령을 실행합니다.

  • 글을 작성하고 확인해 봅니다.
  • 글을 작성하고 저장한 후 아래의 명령을 실행하면 바로 Github 저장소로 배포됩니다.
